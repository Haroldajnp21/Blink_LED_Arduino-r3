# Proyecto Blink_LED Arduino

Este repositorio contiene un proyecto desarrollado en **C++** utilizando **ARDUINO IDE**.  
El objetivo es fomentar la colaboración entre desarrolladores, mantener buenas prácticas de programación y documentar claramente el proceso de desarrollo.

---

## 📌 Características
- Lenguaje principal: **C++**
- IDE: **ARDUINO IDE**
- Arquitectura modular y escalable
- Uso de control de versiones con **Git**
